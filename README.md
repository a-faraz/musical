# spotified
A react app which uses spotify's api to look up artists.

### Includes:

- react
- react-router
- material-ui
- babel
- webpack


### Getting Started:
```
$ git clone https://github.com/a-faraz/spotified
$ cd musical
$ npm install
$ npm start
```

##### Then open [localhost:5000](http://localhost:5000/) to view the app.
