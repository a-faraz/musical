# react-starter-pack
A lightweight boilerplate to get started building react applications.

### Includes:
- Babel
- Webpack

### Getting Started:
```
$ git clone https://github.com/a-faraz/react-starter-pack new-project
$ cd new-project
$ npm install
$ npm start
```

##### Then open [localhost:5000](http://localhost:5000/) to view your app.
